# wp-primordial
Basic WordPress plugin. Remove some unneeded stuff from the html header.

To install with composer:

    "repositories": [
      {
        "type": "vcs",
        "url": "https://github.com/tobiv/wp-primordial.git"
      }
    ],
    "require": {
      "tobiv/wp-primordial": "dev-master"
    }
